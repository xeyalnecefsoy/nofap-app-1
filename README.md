# Ascend - NoFap & Addiction Recovery PWA

[![Netlify Status](https://api.netlify.com/api/v1/badges/YOUR_NETLIFY_SITE_ID/deploy-status)](https://app.netlify.com/sites/YOUR_SITE_NAME/deploys)

A minimalist, high-performance Progressive Web App (PWA) designed to help users overcome addiction through mindfulness, data tracking, and instant redirection tools.

## Features
- **Sobriety Counter:** Tracks days, hours, minutes, and seconds.
- **Rank System:** "Turkic Warrior" themed ranks (e.g., Oyanış, Mübariz, Xaqan).
- **Panic Button:** Instant 4-7-8 breathing exercise to calm urges.
- **Relapse Wizard:** A compassionate but firm reset flow to analyze slips.
- **Daily Motivation:** Curated quotes from history's strongest minds.
- **Background Mode:** Persistent notification tracker using Media Session API.
- **Multilingual:** Full support for English and Azerbaijani (Azərbaycan dili).
- **PWA:** Installable on iOS and Android (offline support included).

## Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **PWA:** next-pwa, Service Workers
